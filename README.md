## Getting Started

Each folder of this repo is dedicated to a single Go fundamental or use case.

### Prerequisites

- Install [Go](https://go.dev/doc/install)

### Running a project locally

Choose the project of the concept/use case you want to run/test:

```
cd [folder-name]
```

Run 

```
go run <file-name>
```
